Description:
A console-based Temperature Converter built in Java that supports five temperature scales — Celsius, Fahrenheit, Kelvin, Rankine, and Reaumur.
Users can convert any single temperature value to all other units at once, or use the custom mode to pick a specific from-unit and to-unit conversion.
All results are displayed with 4 decimal places for precision. Input validation handles invalid units and negative Kelvin values.

Conclusion:
This project demonstrates clean separation of logic and UI in Java — the Converter class handles all math while Main handles all input and display. 
It shows how to use switch statements for mapping unit combinations, printf for formatted output, and how to design a utility-style class in Java. 
A simple but practical project that is great for practising method design, input handling, and unit-based logic in Java.
